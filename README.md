FocusPilot 🚀
A productivity dashboard built with vanilla HTML, CSS, and JavaScript — designed to help you stay focused, track your tasks, and manage your time effectively.

🌟 About the Project
I've been learning full-stack web development for the last 4 months. Things were going great until my end-semester exams forced me to take a break for almost a month. 
When I came back, I felt completely lost — most of the concepts felt blurry and unfamiliar.
Instead of feeling bad about it, I made a decision: revise everything from scratch and build something real.
So I asked myself — what do I actually need in my daily life?
A productivity tool. Something I'd use every day.
And that's how FocusPilot was born. 🎯

✨ Features
📊 Dashboard

Live clock with real-time updates
Live date display
Task stats — total tasks, completion percentage, overdue count
All stats update dynamically as you add, complete or delete tasks

✅ To Do List

Add, complete and delete tasks
Tasks persist across page reloads using localStorage
Completion status saved permanently
Dashboard stats sync automatically

⏱️ Pomodoro Timer

Three modes — Pomodoro (25 min), Short Break (5 min), Long Break (15 min)
Start, Stop and Restart controls
Color changes based on active mode
Alarm sound notification when timer ends
Fully customizable durations from Settings

🎯 Goals

Set and track your monthly goals
Mark goals as completed or delete them
Goals persist using localStorage

📝 Notes

Distraction-free textarea for writing thoughts, ideas or reminders
Auto-saves everything as you type using localStorage
Your notes are always there when you come back

💬 Motivation Quotes

Fetches random quotes from an external API
New Quote button to get fresh motivation anytime
Beautiful glowing quote card design

⚙️ Settings

Theme switcher — switch between dark and light theme
Set your name — personalize the greeting on the topbar
Pomodoro timer duration — customize focus, short break and long break durations
Clear all data — clear tasks or goals separately
Sound toggle — turn timer alarm sound on or off
All settings saved to localStorage
